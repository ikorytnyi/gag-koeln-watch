# gag-koeln-watch

Моніторинг нових оголошень оренди квартир на https://www.gag-koeln.de/immobiliensuche/wohnung-mieten

- `check.py` — завантажує сторінку, парсить оголошення, порівнює з `seen.json`, друкує звіт про нові оголошення (групування за кількістю кімнат + деталі 5-кімнатних) якщо є зміни, оновлює `seen.json`.
- `seen.json` — стан: усі оголошення, побачені на попередньому запуску (по object ID з URL).

Керується хмарною рутиною Claude (`/schedule`), запуск о 8:00, 10:00, 12:00, 14:00 (Europe/Berlin).
